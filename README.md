# Eleven IX — 3D Business Website

A premium 3D interactive website for **Eleven IX** built with Three.js.

## 🚀 Deploy to Render (Free Static Hosting)

### Step 1 — Push to GitHub
1. Create a free account at [github.com](https://github.com)
2. Create a **new repository** called `elevenix-website`
3. Upload all files from this folder to the repo (drag & drop in GitHub UI)

### Step 2 — Connect to Render
1. Go to [render.com](https://render.com) and sign up free
2. Click **"New +"** → **"Static Site"**
3. Connect your GitHub account
4. Select the `elevenix-website` repository
5. Set these settings:
   - **Name**: `elevenix` (your URL will be `elevenix.onrender.com`)
   - **Branch**: `main`
   - **Build Command**: *(leave empty)*
   - **Publish Directory**: `.` (just a dot)
6. Click **"Create Static Site"**
7. Wait ~1 minute → your site is LIVE! 🎉

### Step 3 — Custom Domain (Optional)
1. In Render dashboard → go to your site → **Settings** → **Custom Domains**
2. Add your domain and follow the DNS instructions
